# **App Name**: EduLeap

## Core Features:

- User Authentication: Secure JWT-based authentication system with role-based access control (admin, teacher, student).
- Course Management: Admin panel for creating, updating, and managing courses, including assigning teachers and setting seat limits.
- Content Delivery: Teachers can upload course videos and assign tasks to students through their dashboard.
- Student Dashboard: Students can view ongoing courses, teacher details, course-specific videos, and track their enrollment status.
- Personalized Learning Path: AI-powered tool to suggest relevant courses to students based on their learning history and goals, as tracked in the database, for an adaptive learning experience.
- Consent Management: Admin review and approval for student consent forms, enabling group sessions upon approval.

## Style Guidelines:

- Primary color: Soft lavender (#7C6075) for a calming educational feel.
- Background color: Light off-white (#F9F7F8) for a clean and modern look.
- Accent color: Muted gold (#D6B57C) to highlight key elements.
- Headline font: 'Playfair', a modern serif, for an elegant and high-end feel. Body font: 'PT Sans' sans-serif for body text.
- Use simple, professional icons related to learning and education.
- Clean and professional layout with consistent spacing and typography for enhanced readability.
